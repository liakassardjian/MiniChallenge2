# NanoChallenge2

Esse projeto foi desenvolvido com o intuito de aprimorar nossas habilidades com UIKit. Um _storyboard_ foi desenvolvido em Swift para exemplificar o funcionamento deste aplicativo.
O tema do projeto é __cores__, então aplicamos nossos conhecimentos sobre o assunto no aplicativo. Optamos por trabalhar com o conceito de cor média de uma imagem e cores harmônicas; além disso, expandimos nossa abordagem para a visão de um daltônico. Sendo assim, o aplicativo obtém a cor média de uma imagem, exibe as cores correspondentes à harmonia escolhida e exibe as mesmas cores na visão de um daltônico.
Nessa versão, ainda não foram implementados os algoritos de análise de cores, uma vez que o intuito é trabalhar com interface e não com o programa em si.
